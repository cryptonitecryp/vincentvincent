# Track2Earn
On-chain delivery tracking with instant payment settlement using Stellar.

## Problem
A sari-sari store owner in Metro Manila cannot verify deliveries, leading to disputes and lost income.

## Solution
Track deliveries using proof hashes and release payments instantly via Stellar smart contracts.

## Timeline
Day 1–2: Contract + tests  
Day 3–4: Simple frontend (QR + submit)  
Day 5: Demo + polish  

## Stellar Features Used
- Soroban smart contracts
- XLM transfers
- Trustlines

## Vision
Bring trust and instant payments to informal delivery economies.

## Prerequisites
- Rust
- Soroban CLI
